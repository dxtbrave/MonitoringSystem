# 风机监测系统可视化展示
## 本项目由vuecli3搭建，技术栈包括VUE2+Echarts+axios+Mock.js+ElementUI。

## 项目内容包括：

1.1920*1080内容

2.风机发电量占比（圆环图）

3.风机负载（圆环图）

4.影响健康指标分布图（南丁格尔玫瑰图）

5.监测设备状态（柱状图）

6.温度/湿度监测（柱状图）
